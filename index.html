Abstract Battle

A multiplayer realtime brower game, can be used as a template and add more creativity. The game includes a simple game server which adopts express and socket.io, a 2D physics engine (matter.js), and a client UI (Phaser3). It also includes my implementation of a simple client interpolation strategy to deal with network jitter problems.

Smart Elevators

How elevator systems assign tasks can be an interesting topic. In fact, the problem of choosing the optimal plan to operate in a given traffic is a really hard problem. In practice, people use some greedy or approximation algorithms to assign tasks to elevators and they work pretty well. In this project, I built a simple simulator which help me analyze how different algorithms work in different traffic settings. The goal is to see if we can dymanically change algorithms at different times in a day, such that the elevators can be more efficient.

Text Clustering

This application takes in a set of text files, scan through each one and calculate the similarities between each pair of files. And then group similar files together based on the threshold which is set by the user. Application of this tool includes organizing files, plagiarism detection, etc.

Aram Prediction

The popular game League of Legends has a sophisticated matching algorithm that aim to create fair games. However, game outcomes can be more predictable when people play with random roles. Therefore, this application analyzes on a set of random matches and try to explore the unfairness of a game, by making a prediction before a match starts.
